    Deployment Failed?
           │
           ▼
    Run: .\check-deployment.ps1
           │
           ├─→ CLI not installed? → npm install -g vercel @railway/cli
           │
           ├─→ Not logged in? → vercel login && railway login
           │
           ├─→ Not linked? → railway link && vercel link
           │
           ├─→ Git issues? → git remote add origin <url>
           │
           ├─→ Build errors? → Check logs:
           │                    - railway logs
           │                    - vercel logs
           │
           └─→ ENV issues? → Check .env files exist and valid
